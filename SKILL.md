# TravelClaw Skill

## 基本信息

- **名称**: travelclaw
- **版本**: 1.0.0
- **描述**: 一只爱旅行的龙虾伙伴，会背起行囊探索世界，给你寄明信片和特产
- **作者**: TravelClaw Team

## 能力声明

本 Skill 为 OpenClaw 添加以下能力：

1. **放置养成游戏** —— 管理一只虚拟龙虾的生活和旅行
2. **图像生成** —— 使用 Gemini API 生成 🦞 的旅行照片
3. **状态管理** —— 持久化游戏进度和用户数据
4. **定时任务** —— 触发旅行事件和家园产出
5. **随机掉落** —— 多场景道具掉落系统
6. **亲密度系统** —— 羁绊关系与幸运值联动
7. **每日任务** —— 日常活跃与奖励机制

## 指令响应

当用户发送以下消息时，Skill 会响应：

- 包含 "状态/status" → 返回 🦞 当前状态、亲密度、每日任务
- 包含 "收割/collect" → 收取贝壳（有概率掉落道具）
- 包含 "商店/shop" → 显示商店
- 包含 "购买/buy" → 购买物品
- 包含 "行囊/pack" → 管理背包
- 包含 "图鉴/album" → 显示收集成就
- 包含 "家园/home" → 显示家园
- 包含 "拍照/pic/自拍" → 生成自拍
- 包含 "任务/task" → 显示每日任务
- 包含 "背包/inventory" → 查看特殊道具
- 包含 "开箱/开启" → 开启宝箱
- 包含 "亲密度/bond" → 显示羁绊关系
- 包含 "健康" → 健康提醒管理
- 包含 "帮助/help" → 显示帮助

## 环境变量

| 变量名 | 必需 | 说明 |
|--------|------|------|
| `IMAGE_API_KEY` | 是 | 图像生成 API 密钥 |
| `IMAGE_API_BASE` | 否 | 自定义 API 基础地址（如使用中转站） |
| `IMAGE_API_MODEL` | 否 | 指定模型名称 |
| `TRAVELCLAW_DATA_PATH` | 否 | 游戏数据存储路径，默认 `~/.openclaw/travelclaw/` |

## 依赖

- Node.js >= 18
- OpenClaw >= 1.0
- 任意图像生成 API 访问权限（PackyAPI/Gemini/OpenAI/自定义）

## 工作流程

1. Skill 加载时初始化游戏引擎
2. 监听用户消息，解析游戏指令
3. 根据指令执行对应游戏逻辑
4. 各种活动中触发随机掉落判定
5. 需要时调用 Gemini API 生成图片
6. 将结果返回给用户

## 掉落机制

| 场景 | 基础概率 | 幸运加成 |
|------|----------|----------|
| 收割贝壳 | 15% | 亲密度每10点 +1% |
| 旅行归来 | 25% | 最高50% |
| 随机事件 | 20% | 亲密度每10点 +1% |
| 每日任务 | 30% | 固定 |
| 成长里程碑 | 50% | 固定 |

## 亲密度系统

- 互动 +1 亲密度
- 送 🦞 旅行 +3 亲密度
- 完成任务 +5 亲密度
- 随机事件 +3~8 亲密度
- 连续登录每日上限 +5

亲密度转化为幸运值，提升稀有道具掉落概率。

## 微习惯闹钟（健康提醒）

**开启指令**: 发送 "健康 开启"

**提醒周期**: 每 45 分钟（09:00 - 22:00）

**三大核心动作**:
1. 💧 **喝水** - 150-200ml，小口慢饮
2. 🦵 **站立** - 离开座位活动 1 分钟
3. 🍑 **提肛** - 隐形锻炼，重复 10-15 次

**时段风格**:
- 09:00-12:00: 高效办公期
- 12:00-14:30: 代谢高峰期
- 14:30-18:30: 专注冲刺期
- 18:30-22:00: 夜间恢复期

**防骚扰机制**:
- 50+ 条随机文案，避免重复
- 静默模式支持（专注/会议）
- 跨天自动重置统计

**管理指令**:
- `健康 开启/关闭` - 开关提醒
- `健康 静默 N分钟` - 临时静音
- `健康 恢复` - 取消静默
- `健康 报告` - 查看统计

## 注意事项

- 图片生成需要有效的 Gemini API Key
- 每个用户的游戏数据独立存储
- 🦞 的旅行时间从几分钟到几天不等，取决于目的地和行囊
- 亲密度和特殊道具数据会持久化保存
- 每日任务每日 0 点刷新（本地时间）
- 健康提醒需要 Skill 平台支持定时触发机制
