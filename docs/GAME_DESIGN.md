# 🦞 TravelClaw 游戏设计文档

> 一只爱旅行的龙虾，一段温暖的放置养成之旅

---

## 📖 产品概述

**TravelClaw** 是一个为 OpenClaw 设计的放置养成 Skill。玩家在 IM 应用中养育一只爱旅行的龙虾 🦞，它会自主地背起行囊探索世界，从各地给玩家寄回明信片和特产。

### 核心理念

- **不打扰的陪伴** —— 每天只需几分钟，不会占用玩家大量时间
- **期待与惊喜** —— 🦞 有自己的想法，玩家无法强制，只能等待和期待
- **情感连接** —— 通过长期陪伴建立感情，看着 🦞 从小虾成长为虾王
- **收集与成就** —— 收集世界各地的特产，完成图鉴

---

## 🎮 核心玩法循环

```
收取贝壳 → 购买道具 → 准备行囊 → 🦞 出发旅行
                                          ↓
布置家园 ← 获得特产 ← 🦞 归来 ← 寄回明信片
```

### 日常流程

1. **早晨** —— 打开 IM，收取家园产出的贝壳
2. **白天** —— 查看 🦞 状态，准备旅行物品
3. **出发** —— 🦞 背起行囊，消失几小时到几天
4. **等待** —— 期间可能收到明信片
5. **归来** —— 🦞 带着特产回家，休息后可再次出发

## 🦞 成长系统

### 五个成长阶段

| 阶段 | 天数 | 外观特征 | 旅行范围 | 行为特点 |
|------|------|----------|----------|----------|
| 🦐 **幼虾期** | 0-30 | 半透明小身体，小钳子 | 附近海滩 | 胆小好奇，不敢走远 |
| 🦞 **少年期** | 1-3月 | 颜色变深，钳子变大 | 附近城市 | 开始探索，偶尔迷路 |
| 🦞 **青年期** | 3-6月 | 鲜艳的橙红色，强壮双钳 | 国内长途 | 自信满满，喜欢冒险 |
| 🦞 **壮年期** | 6-12月 | 巨大威武的钳子，霸气 | 国际旅行 | 见多识广，带回稀有物品 |
| 👑 **虾王期** | 1年+ | 金色纹路，传奇般存在 | 全球冒险 | 传奇存在，遇到神秘事件 |

### 成长机制

- **时间积累** —— 从创建角色开始计算天数
- **互动加成** —— 玩家每天互动可略微加速成长
- **阶段解锁** —— 每个阶段解锁新的地点和道具

### 成长纪念

每当 🦞 升级到新阶段，会：
- 生成一张成长纪念照片
- 解锁新的旅行目的地
- 可能学会新的行为（如与其他海洋生物互动）

## 💰 经济系统

### 货币：贝壳 🐚

**获取方式：**
1. **家园产出** —— 每小时自动产出 0-3 个贝壳
2. **旅行带回** —— 🦞 有时会带回贝壳
3. **出售特产** —— 多余的特产可以出售

**用途：**
- 购买食物、装备、装饰品
- 扩建行囊容量（未来功能）
- 解锁特殊地点（未来功能）

### 道具分类

#### 🍽️ 食物类

| 道具 | 价格 | 效果 |
|------|------|------|
| 海藻沙拉 | 10🐚 | 基础食物，恢复体力 |
| 浮游生物团 | 15🐚 | 轻便，减少旅行时间 |
| 死鱼 | 25🐚 | 🦞 最爱，增加幸运值 |
| 热泉硫化物 | 50🐚 | 高级食物，大幅增加能量 |
| 沉船饼干 | 80🐚 | 古老食物，带来好运 |

**食物机制：**
- 决定 🦞 能旅行的最长时间
- 影响旅行归来时的状态
- 饿着肚子出发会不开心

#### 🎒 装备类

| 道具 | 价格 | 效果 |
|------|------|------|
| 褪下的旧壳 | 30🐚 | 护身符，增加幸运值 |
| 强壮的右钳 | 60🐚 | 可以打开坚硬贝壳 |
| 潜水镜 | 45🐚 | 解锁深海目的地 |
| 小划水桨 | 40🐚 | 减少旅行时间 |
| 防水地图 | 100🐚 | 增加发现新地点概率 |
| 指南针 | 55🐚 | 可以去更远的地方 |

**装备策略：**
- 不同组合产生不同结果
- 装备影响可到达的目的地
- 某些特产需要特定装备才能获得

#### 🏠 家园装饰类

纯视觉装饰，不影响游戏数值，但：
- 让家园更温馨
- 在自拍中展示个性
- 某些装饰可能有隐藏效果（彩蛋）

装饰品包括：珊瑚枝、海葵、沉船舵轮、船锚、漂流瓶、渔网、贝壳风铃等。

## ✈️ 旅行系统

### 行囊机制

- **容量限制** —— 最多 5 个物品槽位
- **策略搭配** —— 食物 + 装备的组合影响旅行结果
- **出发后锁定** —— 🦞 带走行囊，期间不能更改

### 旅行流程

```
准备行囊 → 点击出发 → 计算目的地和时间 → 🦞 消失
                                               ↓
玩家收到通知 ← 生成明信片 ← 到达目的地 ← 旅行中
                                               ↓
🦞 在家休息 ← 玩家收到通知 ← 🦞 归来 ← 旅行结束
```

### 目的地系统

#### 五大区域

| 区域 | 解锁阶段 | 特点 | 地点示例 |
|------|---------|------|----------|
| **附近海滩** | 幼虾期 | 安全、短途 | 阳光沙滩、礁石海岸、潮间水洼 |
| **附近城市** | 少年期 | 人文气息 | 渔港、鱼市场、海滨公园 |
| **国内名胜** | 青年期 | 风景名胜 | 长城、西湖、漓江、黄山 |
| **国际旅行** | 壮年期 | 世界地标 | 埃菲尔铁塔、圣托里尼、富士山 |
| **全球冒险** | 虾王期 | 神秘地点 | 南极、马里亚纳海沟、亚特兰蒂斯 |

#### 地点特性

每个地点有：
- **独特景观** —— AI 生成图片的背景
- **特色活动** —— 🦞 在那里的行为
- **特产列表** —— 可能带回的物品
- **稀有度** —— 某些地点需要幸运值或装备才能到达

### 时间系统

- **旅行时长** —— 30分钟到24小时不等
- **受因素影响** —— 食物、装备、目的地距离
- **不可加速** —— 放置游戏的核心体验

### 明信片系统

**触发时机：**
- 到达新地点时
- 遇到特殊事件时
- 发现稀有物品时

**明信片内容：**
- 🦞 在当地的 AI 生成照片
- 一段手写风格的留言
- 位置信息和时间戳

## 🧠 个性化系统

### 性格学习机制

🦞 会通过分析玩家的消息来学习性格偏好：

```
玩家消息 → 关键词提取 → 性格分数更新 → 影响图片生成
```

### 性格类型

| 类型 | 关键词 | 对 🦞 的影响 |
|------|--------|-------------|
| **可爱系** | 可爱、萌、宝贝 | 表情更萌，动作更活泼 |
| **沉稳系** | 沉稳、优雅、淡定 | 举止优雅，动作更慢 |
| **威武系** | 威武、霸气、钳子大 | 钳子更突出，姿态更霸气 |
| **好奇系** | 好奇、探险、调皮 | 眼睛更亮，喜欢探索姿势 |
| **慵懒系** | 懒、悠闲、佛系 | 懒散姿势，眯着眼睛 |

### 个性化表达

- **图片生成** —— Prompt 中加入性格修饰词
- **文案风格** —— 不同性格的 🦞 说话方式不同
- **行为偏好** —— 影响旅行目的地选择（未来功能）

## 🎨 美术风格

### 视觉风格

- **手绘插画** —— 温暖、治愈、有质感
- **柔和色彩** —— 避免高饱和度
- **Q版比例** —— 可爱但不失龙虾特征

### 🦞 角色设计要点

**基础特征：**
- 橙红色外壳（幼虾期偏透明）
- 两只不对称的大钳子
- 两根触须
- 黑色圆眼睛

**各阶段差异：**
- 幼虾：半透明、小钳子、大眼睛
- 少年：颜色加深、钳子开始发育
- 青年：鲜艳橙红、强壮双钳
- 壮年：巨大钳子、霸气姿态
- 虾王：金色纹路、传奇光芒

### 图片生成策略

**Prompt 结构：**
```
[角色描述] + [阶段特征] + [性格修饰] + [场景描述] + [风格修饰]

示例：
"A cute red lobster character, young adult size with bright orange shell 
and growing claws, curious and playful expression with wide eyes, 
at Eiffel Tower in Paris, looking up in wonder, illustration style, 
hand-drawn, cozy atmosphere"
```

## 🔧 技术架构

### 数据流

```
用户 IM (WhatsApp/Telegram/Slack)
    ↓
OpenClaw Gateway
    ↓
TravelClaw Skill
    ├─ 指令解析
    ├─ 游戏逻辑 (GameEngine)
    ├─ 状态更新 (StateManager)
    └─ 图片生成 (ImageClient - 多平台支持)
    ↓
回复 + 图片
    ↓
用户 IM
```

```json

    "birth_date": "2026-03-11",
    "total_days": 45,
    "traits": ["cute", "curious"],

  "inventory": {
    "shells": 128,
    "items": { "seaweed_salad": 3 },
    "backpack": [],

  },
  "gameState": {
    "state": "at_home",
    "travel_start": null,
    "travel_end": null,
    "current_location": null
  }

```

### 模块化设计

```
src/
  core/           # 游戏核心逻辑
    Claw.js       # 角色成长、个性化
    Inventory.js  # 库存、经济
    GameEngine.js # 游戏循环、状态管理
  content/        # 游戏内容数据
    items.js      # 道具定义
    locations.js  # 地点定义
    dialogues.js  # 文案、性格学习
  image/          # 图片生成
    ImageClient.js    # 统一客户端
    PackyClient.js    # PackyAPI 实现

    PromptBuilder.js  # Prompt 构建
  storage/        # 数据持久化
    StateManager.js
```

## 🎯 设计原则

### 1. 极简主义

- 不做复杂操作，降低认知负担
- 每天几分钟即可完成日常
- 不惩罚不活跃玩家

### 2. 情感优先

- 文案温暖治愈
- 🦞 有自己的"想法"
- 创造牵挂感和期待感

### 3. 渐进式复杂度

- 新手期简单明了
- 随着成长逐渐解锁新功能
- 不一次性给玩家太多信息

### 4. 开放性与扩展性

- 支持多种图片生成 API
- 易于添加新地点和道具
- 模块化代码结构
