# 🦞 TravelClaw 设计文档

## 产品概述

TravelClaw 是一个为 OpenClaw 设计的放置养成 Skill，让玩家在 IM 应用中养育一只爱旅行的龙虾 🦞。

## 核心理念

### 情感连接
- **牵挂感** —— 🦞 有自己的生活节奏，玩家无法强制控制
- **期待感** —— 不知道 🦞 什么时候会寄来明信片
- **惊喜感** —— 收到意想不到的旅行照片和特产
- **成就感** —— 收集完整的图鉴，看着 🦞 成长

### 放置体验
- 不需要时刻在线
- 每天几分钟即可
- 不会错过重要内容
- 适合忙碌的现代生活

## 游戏系统

### 1. 成长系统

| 阶段 | 天数 | 特征 | 旅行范围 |
|------|------|------|----------|
| 🦐 幼虾期 | 0-30 | 半透明，小钳子 | 附近海滩 |
| 🦞 少年期 | 1-3月 | 颜色变深，钳子变大 | 附近城市 |
| 🦞 青年期 | 3-6月 | 橙红色，强壮双钳 | 国内长途 |
| 🦞 壮年期 | 6-12月 | 巨大钳子，霸气 | 国际旅行 |
| 👑 虾王期 | 1年+ | 金色纹路，传奇 | 全球冒险 |

### 2. 经济系统

- **贝壳** —— 基础货币
  - 家园自然产出
  - 旅行带回
  - 特产出售

- **道具** —— 影响旅行
  - 食物：恢复体力，影响旅行时长
  - 装备：影响目的地，增加幸运值
  - 装饰：纯视觉，提升幸福感

### 3. 旅行系统

```
准备行囊 → 出发 → 旅行中 → 寄明信片 → 归来 → 休息 → 可再次出发
```

**行囊策略**：
- 最多 5 个物品
- 食物决定旅行时长
- 装备影响目的地和收获
- 不同组合产生不同结果

### 4. 个性化系统

🦞 会学习玩家的互动方式：

```
玩家输入 → 关键词提取 → 性格分数更新 → 影响图片生成
```

**性格类型**：
- 可爱系（萌、宝贝、心疼）
- 沉稳系（沉稳、优雅、淡定）
- 威武系（霸气、强壮、钳子大）
- 好奇系（好奇、探险、调皮）
- 慵懒系（懒、悠闲、佛系）

## 技术架构

### 数据流

```
用户消息
    ↓
OpenClaw Gateway
    ↓
TravelClaw Skill
    ├─→ 指令解析
    ├─→ 游戏逻辑
    ├─→ 状态更新
    └─→ 图片生成 (Gemini API)
    ↓
回复 + 图片
    ↓
用户 IM
```

### 状态存储

```json
{
  "claw": {
    "birth_date": "2026-03-11",
    "total_days": 15,
    "traits": ["cute", "curious"],
    "personality_scores": { ... }
  },
  "inventory": {
    "shells": 128,
    "items": { "seaweed_salad": 3 },
    "backpack": [...],
    "souvenirs": { ... }
  },
  "gameState": {
    "state": "traveling",
    "travel_start": "...",
    "travel_end": "...",
    "current_location": "eiffel_tower"
  }
}
```

### 图片生成流程

```
1. 构建 Prompt
   - 角色描述（阶段特征 + 性格修饰）
   - 场景描述（地点 + 活动）
   - 风格修饰（手绘插画）

2. 调用 PackyAPI (豆包 Seedream)
   - model: gemini-2.0-flash-exp-image-generation
   - 返回 base64 图片数据

3. 发送给用户
   - 图片 + 明信片文案
```

## 用户交互

### 指令列表

| 指令 | 功能 |
|------|------|
| `状态` | 查看 🦞 当前状态 |
| `收割` | 收取贝壳 |
| `商店` | 购买道具 |
| `行囊` | 管理旅行背包 |
| `出发` | 开始旅行 |
| `图鉴` | 查看收集成就 |
| `家园` | 装饰小窝 |
| `拍照` | 生成自拍 |
| `帮助` | 显示指令列表 |

### 自然语言交互

🦞 会回应玩家的自然语言：

- "你好可爱" → "🦞 听到夸奖开心地挥舞钳子！"
- "什么时候回来" → 显示旅行进度
- "记得吃饭" → "会记得吃背包里的食物~"
